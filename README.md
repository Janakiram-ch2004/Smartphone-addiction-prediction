# Smartphone Addiction Predictor

A Machine Learning web application that predicts smartphone addiction risk based on a user's smartphone usage behavior. The application is built using Python, Scikit-learn, and Streamlit, and is deployed on Streamlit Cloud.

## Live Demo

https://smartphone-addiction-prediction-gbhouxoqgnlnih5poc3bsp.streamlit.app/

## GitHub Repository

https://github.com/ch-janakiram2004/Smartphone-addiction-prediction

---

## Project Overview

Smartphone addiction has become a growing concern in today's digital world. This project uses a Machine Learning model to analyze user behavior and predict whether a user is at risk of smartphone addiction.

The application provides instant predictions through an easy-to-use web interface.

## Installation

Clone the repository

```bash
git clone https://github.com/ch-janakiram2004/Smartphone-addiction-prediction.git
```

Move into the project folder

```bash
cd Smartphone-addiction-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
